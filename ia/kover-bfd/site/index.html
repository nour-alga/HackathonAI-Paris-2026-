Les 4 micro-services

Solidity

VaultClient.sol

function emergencyHalt() external onlySecurityBot {
    _pause();
    emit CircuitBreakerTriggered(tx.origin, block.timestamp);
}
  • Pausable + Ownable + ReentrancyGuard
  • Rotation HSM via rotateSecurityBot
  • Errors custom (gas-cheap)
Node.js

mempool_streamer.js

provider.on('pending', async (txHash) => {
  const tx = await provider.getTransaction(txHash);
  if (tx?.to?.toLowerCase() !== TARGET_LC) return;
  producer.send({ topic, messages: [{ value: serialize(tx) }] });
});
  • Backoff exponentiel + jitter (cap 30 s)
  • Watchdog heartbeat 30 s
  • Logger Winston JSON, latence µs
PyFlink

time_series_aggregator.py

parsed.key_by(lambda t: t[0]) \
  .window(TumblingEventTimeWindows.of(Time.seconds(1))) \
  .reduce(_SumReducer(), window_function=_EmitWindow())
  • Watermark bounded-out-of-orderness 500 ms
  • Sink HTTP async — sémaphore bornée 64
  • Validation Pydantic v2 à la frontière Kafka
FastAPI

anomaly_detector.py

if score < THRESHOLD and volume >= DESTRUCTIVE_WEI:
